# Tableflip

**Tableflip** — программа, созданная чтобы помочь моему другу с его работой

## Возможности

- Извлечение персональных данных людей из таблиц в Word-файлах
- Массовое редактирование вводимых полей
- Автоматическое заполнение форм на web-странице
- Система лицензирования  
  _Присутствует в коде, но в текущей версии не используется_

## Требования

Для корректной работы программе необходимы:

- `chromedriver-win64`
- `chrome-win64`

Официальные версии можно скачать здесь:  
https://googlechromelabs.github.io/chrome-for-testing/
